================================================================================
                    SHIPPING API CREDENTIALS REQUIREMENTS
================================================================================

This document explains the API credentials needed for each shipping integration 
option.

================================================================================
                         OPTION 1: EASYSHIP
================================================================================

WHAT YOU NEED TO GET:
---------------------
Visit: https://www.easyship.com

1. Create an Easyship account
2. Verify your business
3. Get your API Key from dashboard

CREDENTIALS NEEDED:
-------------------
• Easyship API Key
• Default shipper address (for returns)

COST: Free plan (50 shipments/month), then paid plans from $19/month

--------------------------------------------------------------------------------
                         OPTION 2: CLICKPOST
================================================================================

WHAT YOU NEED TO GET:
---------------------
Visit: https://www.clickpost.ai

1. Contact their sales team
2. Sign up for merchant account
3. Get API credentials

CREDENTIALS NEEDED:
-------------------
• ClickPost API Key
• ClickPost API Secret
• Merchant ID

COST: Contact sales for pricing

--------------------------------------------------------------------------------
                         OPTION 3: DIRECT APIs (ARAMEX + NAQEL)
================================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
                        ARAMEX API
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WHAT YOU NEED TO GET:
---------------------
Visit: https://www.aramex.com/developers-solution-center/aramex-apis

1. Contact Aramex business development team
2. Apply for API access with your business details
3. Receive credentials after approval

CREDENTIALS NEEDED:
-------------------
Field                   | Description
------------------------|-------------------------------------------
Account Number          | Your Aramex account number
User Name               | API username
Password                | API password
Account PIN             | Security PIN for API calls
Shipper Name            | Your company name (for shipping labels)
Shipper Phone           | Warehouse phone number
Shipper Email           | Warehouse email
Shipper Address         | Warehouse street address
Shipper City            | Warehouse city (e.g., Dubai)
Shipper Country         | Warehouse country code (e.g., AE)

HOW TO GET:
-----------
1. Go to: www.aramex.com/developers-solution-center
2. Click "Request API Access"
3. Fill business application form
4. Aramex team will contact you within 2-3 business days
5. Receive test credentials first, then production access

COST: No monthly fee, pay per shipment (~15-50 AED depending on weight)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
                        NAQEL EXPRESS API
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WHAT YOU NEED TO GET:
---------------------
Visit: https://www.naqelexpress.com

1. Contact Naqel business/sales team
2. Apply for merchant account with API access
3. Receive credentials after business verification

CREDENTIALS NEEDED:
-------------------
Field                   | Description
------------------------|-------------------------------------------
User Name               | API username
Password                | API password
API Key                 | Unique API key for authentication
Shipper Name            | Your company name
Shipper Phone           | Warehouse phone number
Shipper Address         | Warehouse address
Shipper City            | Warehouse city (e.g., Riyadh)
Shipper Country         | Country code (e.g., SA for Saudi Arabia)

HOW TO GET:
-----------
1. Go to: www.naqelexpress.com
2. Contact their sales team (call or email)
3. Provide business registration documents
4. Request API access for e-commerce integration
5. Receive credentials after approval

COST: No monthly fee, pay per shipment (~15-40 SAR depending on weight)
